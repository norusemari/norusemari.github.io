Learning CSS - part.3

Posted Jan 14 2021-01-14T23:45:00+09:00 by norusemari

3.17 States

크롬 개발자 도구에서 확인할 수 있는 가장 중요한 selector는 아래 화면에서 보이는 Force element state 이다. (active, focus, hover, visited, focus-within)

active

<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 1000vh;
        margin: 50px;
      }
      button:active {
        background-color: tomato;
      }
    </style>
  </head>
  <body>
    <div>
      <button>hello</button>
    </div>
  </body>
</html>

active는 버튼을 마우스로 클릭하고 있는 동안 버튼이 토마토 색으로 변한다. 위의 이미지는 설명을 위해 개발자 도구에서 Force elements state의 :active 를 선택해 두었기 때문에 마우스를 클릭했을때 어떤 모양이 되는지 보여주고 있다.

hover

<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 1000vh;
        margin: 50px;
      }
      button:hover {
        background-color: tomato;
      }
    </style>
  </head>
  <body>
    <div>
      <button>hello</button>
    </div>
  </body>
</html>

hover는 버튼 위에 마우스를 올렸을때 토마토 색으로 변한다. 위의 이미지 역시 설명을 위해 :hover를 체크해서 마우스 오버한 결과를 보여주고 있다. 여기서 중요한 포인트는, button 엘리먼트에 대해서 하나라도 스타일을 바꾸면 처음부터 다시 스타일을 해야한다는 점.

예를들어 위에서는 button:hover { background-color: tomato; } 에서 버튼의 배경색을 바꾸었다. 이렇게되면 기본적으로 button 엘리먼트가 가지고 있던 default 스타일이 모두 풀리고, 원하는 스타일을 직접 지정해 줘야한다는 것

  • [참고] button에 default로 적용된 스타일

focus

focus는 키보드로 선택되었을 때 나타나는 스타일을 뜻한다. 마우스를 클릭하고 있을때와는 정의된 액션이 다르다.

<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 1000vh;
        margin: 50px;
      }
      input:focus {
        background-color: tomato;
      }
    </style>
  </head>
  <body>
    <div>
      <input type="text" />
      <input type="text" />
    </div>
  </body>
</html>

첫번째 input을 마우스로 클릭해두면(선택해 두면) 해당 input이 토마토색으로 변하고, 거기서 키보드 tab키를 누르면 오른쪽 input으로 이동하며 해당 input이 토마토색으로, 첫번째 input은 원래 스타일로 돌아간다.

visited

visited는 링크에만 적용된다. 방문한 링크에 대해 방문기록을 남기는것으로 아래 예시에서는 방문시 링크 텍스트가 토마토 색으로 변하게 만들어 보았다. 검색엔진에서 검색기록이 남아있는 것도 이런 원리.

<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 1000vh;
        margin: 50px;
      }
      a:visited {
        color: tomato;
      }
    </style>
  </head>
  <body>
    <div>
      <a href="https://apple.com">Go to apple</a>
    </div>
  </body>
</html>

focus-within

focus-within은 부모에게 적용되는 개념이다. 즉, focused인 상태를 가진 부모 엘리먼의 상태를 뜻함. (자식들중 하나가 focused되면 부모의 스타일/상태가 설정에 맞춰 변한다는 것)

<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 1000vh;
        margin: 50px;
      }
      form {
        border: 1px solid salmon;
        display: flex;
        flex-direction: column;
        padding: 20px;
      }
      form:focus-within {
        border-color: seagreen;
      }
    </style>
  </head>
  <body>
    <form>
      <input type="text" name="" id="" />
      <input type="text" name="" id="" />
      <input type="text" name="" id="" />
    </form>
  </body>
</html>

form의 border가 salmon색으로 되어있는데, form 안의 input들 중 하나가 focused 되면 form의 border-color가 seagreen색으로 바뀐다.


<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 1000vh;
        margin: 50px;
      }
      form {
        border: 1px solid salmon;
        display: flex;
        flex-direction: column;
        padding: 20px;
      }
      form:hover input {
        background-color: sienna;
      }
    </style>
  </head>
  <body>
    <form>
      <input type="text" name="" id="" />
      <input type="text" name="" id="" />
      <input type="text" name="" id="" />
    </form>
  </body>
</html>

이렇게 form:hover input 으로 state를 연계해서 사용할 수 있으며, 위의 예시는 부모인 form이 hover 상태가 되면 자식인 input의 상태가 바뀌는 예시다. (form에 마우스 오버시, input안의 배경색이 sienna로 바뀜)


<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 1000vh;
        margin: 50px;
      }
      form {
        border: 1px solid salmon;
        display: flex;
        flex-direction: column;
        padding: 20px;
      }
      form:hover input:focus {
        background-color: sienna;
      }
    </style>
  </head>
  <body>
    <form>
      <input type="text" name="" id="" />
      <input type="text" name="" id="" />
      <input type="text" name="" id="" />
    </form>
  </body>
</html>

form:hover input:focus { ~ }

이렇게 state에 대한 설정을 해주면, form위에 마우스가 올라가더라도 임의의 input에 focus되어 있지 않으면 해당 input에 대한 스타일 변화는 없다. 즉 form 위에 마우스도 올라가 있으면서 focus되어 있는 input이 있을때 해당 input의 배경색만 sienna로 바뀌는 설정

3.18 Recap

::placeholder

placeholder를 스타일링 하기 위한 코드 - [링크]

<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      input::placeholder {
        color: yellowgreen;
      }
    </style>
  </head>
  <body>
    <form>
      <input type="text" placeholder="Name" />
    </form>
  </body>
</html>

placeholder 즉, Name이라는 문자만 yellowgreen색으로 칠했다. 입력창 안에 텍스트를 입력해보면 검은색으로 나온다.

::selection

<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      p::selection {
        color: white;
        background-color: yellowgreen;
      }
    </style>
  </head>
  <body>
    <p>
      Lorem ipsum dolor, sit amet consectetur adipisicing elit. Non perferendis,
      ex nesciunt amet aliquid minus alias quis, impedit, omnis aspernatur
      itaque culpa debitis aperiam? Omnis nisi ab mollitia provident pariatur!
    </p>
  </body>
</html>

흰 바탕 검은색 글씨로 이루어져있던 텍스트 위에, 임의의 위치를 클릭해서 드래그 하면 해당 영역만큼 white색의 글씨, yellowgreen색 바탕으로 칠해진다. ::selection{ } 안에 padding: ;은 적용되지 않는다.

::first-letter

참고 자료 [링크]

<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      p::first-letter {
        /* p::first-line의 경우 첫번째 라인에 대해서 적용된다. */
        color: white;
        background-color: yellowgreen;
        font-size: 25px;
      }
    </style>
  </head>
  <body>
    <p>
      Lorem ipsum dolor, sit amet consectetur adipisicing elit. Non perferendis,
      ex nesciunt amet aliquid minus alias quis, impedit, omnis aspernatur
      itaque culpa debitis aperiam? Omnis nisi ab mollitia provident pariatur!
    </p>
  </body>
</html>

나열된 문장의 첫 문자에 대해 명령어가 적용되며, ::first-line{ } 의 경우 첫 번째 줄에 대해서만 스타일이 적용된다.

Recap

  • state
    • :hover{ } : 마우스 커서가 어떤것 위에 올라가 있는 경우
    • :active{ } : 마우스 커서가 어떤것을 클릭하고 있을 때
    • :focus{ } : form과 같은 경우 키보드로 선택할 수 있다. tab키 눌러서 선택
    • :visited{} : 우리가 클릭했거나 방문한 적이 있는 링크에 적용됨
    • :focus_within { } : focus인 상태인 자식을 가진 부모 엘리먼트에 적용

3.19 Colors and Variables

color

  • hexadecimal color (16진수 컬러) : #FCCE00
  • rgb color : rgb(252, 206, 0)
  • rgba : rgb + alpha(투명) : rgb(252, 206, 0, 0.5)
    • 알파값: 최대 1, 최소 0

custom property = variable

:root 를 활용해 마치 프로그래밍 언어에서 변수를 할당하듯이 사용할 수 있다.

  • 네이밍 규칙 : –변수명-변수명 « 시작은 (-) 두개 그이후로 띄어쓰기 대신 (-)하나로 채우기
  • 색상 뿐만 아니라 다양한 설정을 custom property로 할당해서 사용할 수 있다.
  • 사용할 수 있는 브라우저 확인 방법 : [링크]
    • 하단의 표를 보면 internet explorer를 제외하고 모두 적용 가능.
<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      :root {
        /* :root는 기본적으로 모든 document의 뿌리가 된다. */
        /* custom property 라고 부르지만, 니꼬는 variable이라 부르길 선호한다네 */
        --main-color: #fcce00;
        --default-border: 1px solid var(--main-color);
      }
      p {
        background-color: var(--main-color);
      }
      a {
        color: var(--main-color);
        border: var(--default-border);
      }
    </style>
  </head>
  <body>
    <p>
      Lorem ipsum dolor, sit amet consectetur adipisicing elit. Non perferendis,
      ex nesciunt amet aliquid minus alias quis, impedit, omnis aspernatur
      itaque culpa debitis aperiam? Omnis nisi ab mollitia provident pariatur!
    </p>
    <a href="https://google.com"> Go to google </a>
  </body>
</html>
