3.10 Flexbox Part One

<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 100vh; /*viewpoint height:화면에 맞춰 바뀐다, 100vh:화면높이의 100%*/
        margin: 20px;
        display: flex; /*body가 flex container가 된다.*/
        /*justify-content: center;*/
        /*justify-content: space-evenly;*/
        justify-content: space-around;
        align-items: center;
      }
      div {
        width: 300px;
        height: 300px;
        background-color: teal;
      }
      #second {
        background-color: wheat;
      }
    </style>
  </head>
  <body>
    <div></div>
    <div id="second"></div>
    <div></div>
  </body>
</html>

flexbox는 box를 화면의 어느위치에나 둘 수있다는 장점이 있지만 지켜야 하는 몇가지 규칙이 있다.

  • 그 중 규칙 하나, flex 설정은 부모 엘리먼트에만 적어야함. 자식 엘먼트에는 어떤 것도 적지 말아야 한다.
    • 위의 코드 기준, div의 부모인 body를 display:felx; 로 지정한다.
  • flexbox main axis(주축 - 수평축), cross axis(교차축 - 수직축). 교차축은 기본적으로 수직축인데, 교차축과 수직축은 flex-direction 으로 바꿔줄 수 있다.
    • justyfiy-content는 주축(main axis)에 적용
    • align-items는 교축(cross asxis)에 적용

위의 코드에서 body가 div의 부모 역할을 하고 있는데, body의 높이가 div들을 배치한 높이와 같다면(이미 div가 body의 수직 중심축에 있다면) align-items: center; 명령어를 입력해도 화면상 바뀌는게 없어 보인다. 이런경우 body에 height를 충분히 설정해줘서 테스트 해보면 align-items: center; 명령이 수행된 것을 알 수 있다.

body의 heigth를 설정할때 px가 아니나 vh를 사용했는데 이는 사용자가 웹페이지에 접근할때 사용되는 기기의 화면 크기에 맞춰 사이즈가 조정되는 단위이다. 100vh는 내가 사용하고 있는 화면 높이의 100%에 해당한다.

3.11 Flexbox Part Two

위에서 잠시 언급했듯이 fix-direction: column; 코드를 사용하면 기존에 주축과 교차축의 기준을 바꿀 수 있다. 즉 기존 주축이 현재 교차축 방향이 되고 기존 교차축이 현재 주축 방향이 되는거다.

  • fix-direction: row; 가 default 이다.
<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 100vh; /*viewpoint height:화면에 맞춰 바뀐다, 100vh:화면높이의 100%*/
        margin: 20px;
        display: flex; /*body가 flex container가 된다.*/
        flex-direction: column; /*default는 row이다. 이렇게하면 수평/수직축이 바뀜*/
        /*flex-direction: column-reverse;*/
        justify-content: space-around; /*따라서 수평이었던 주축에 적용되던게, 수직인 주축에 적용됨*/
        align-items: flex-end;
      }
      div {
        width: 250px;
        height: 250px;
        background-color: teal;
      }
      #second {
        background-color: wheat;
      }
    </style>
  </head>
  <body>
    <div></div>
    <div id="second"></div>
    <div></div>
  </body>
</html>

flex-direction: ; 를 통해 축의 방향을 바꿀 수 있는데 default는 row이고 그밖에도 row-reverse, column, column-reverse 등이 있으며 이에대한 설명은 해당 링크 참고!

<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 100vh; /*viewpoint height:화면에 맞춰 바뀐다, 100vh:화면높이의 100%*/
        margin: 20px;
        display: flex; /*body가 flex container가 된다.*/
        justify-content: space-between;
        align-items: center;
        flex-direction: row-reverse;
        /*flex-wrap: nowrap;*/ /*화면을 줄여도 크기에 맞춰 너비가 줄어듬*/
        /*flex-wrap: wrap; 화면이 줄면 거기에 맞춰 컨텐츠를 아래로 내림*/
        flex-wrap: wrap-reverse; /*화면이 줄어들때 컨텐츠 움직이는 순서가 바뀜*/
      }

      div {
        width: 250px;
        height: 250px;
        background-color: teal;
        display: flex; /*이렇게 div를 flex container로 만들면 div안의 텍스트가 자식이 되어 적용받는다.*/
        justify-content: center;
        align-items: center;
        font-size: 50px;
        color: rebeccapurple;
      }
      #second {
        background-color: wheat;
      }
    </style>
  </head>
  <body>
    <div>1</div>
    <div id="second">2</div>
    <div>3</div>
  </body>
</html>

여기서는 flex-wrap: ; 를 주의깊게 살펴보자, 화면이 줄어듬에 따라서 최초에 설정해둔 크기(div 의 크기 width)에 상관없이 컨텐츠를 줄어들게 할건지, 아니면 공간이 모자라면 컨텐츠를 아래로 내릴건지를 설정해주는 코드 이다.

위와같이 flex-wrap는 flex-direction과 함께 쓸수 있으며, 아래 이미지를 통해 화면 사이즈가 줄어듬에 따라 어떻게 컨텐츠가 움직이는지 확인해 볼 수 있다.

  • flex-wrap에 대한 설정은 해당 링크 확인!

flexbox는 더 많은 property가 있으나 우선은 여기까지!

3.12 Position : Fixed

position은 레이아웃보다 위치를 원하는 곳으로 위로, 옆으로 옮기고 싶을때 사용

  • position: fixed; 는 스크롤 해도 항상 지정된 위치에 머무르게 하며
  • 다른 레이아웃에 상관없이 항상 가장 위에 위치하게 된다.
  • top, left, right, bottom 으로 original position로부터 위치 조정 가능
<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 1000vh;
        margin: 20px;
      }
      div {
        width: 300px;
        height: 300px;
        background-color: teal;
      }
      #different {
        position: fixed; /*기존에 레이아웃에 상관없이 지정된 위치에 고정 & 제일 위로 노출*/
        top: 100px; /*웹페이지에서 위에서 100px 아래 위치*/
        left: 200px; /*웹페이지에서 왼쪽에서 200px 떨어져서 위치*/
        background-color: wheat;
        width: 350px;
      }
    </style>
  </head>
  <body>
    <div></div>
    <div id="different"></div>
  </body>
</html>

position: fixed; 를 통해 스크롤을 위아래로 움직여도 지정한 위치에 고정되어있는 div 박스를 확인할 수 있다.

3.13 Position : Relative & Absolute

position : relative; 는 처음 위치를 기준으로 top, left, right, bottom 설정에 맞춰 위치를 조정한다. 아래 예시는 div 안에 div가 있을 경우, div를 감싸고 있는 div가 안에 있는 div의 처음 위치가 되는 개념!

  • CSS position에 대한 추가 설명은 링크 참고!
<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 1000px;
        margin: 50px;
      }
      div {
        width: 300px;
        height: 300px;
        background-color: wheat;
      }
      .green {
        background-color: teal;
        position: relative;
        width: 100px;
        height: 100px;
        top: -10px;
        left: -10px;
      }
    </style>
  </head>
  <body>
    <div>
      <div class="green"></div>
    </div>
  </body>
</html>


position: absolute; 는 position: relative; 인 부모를 기준으로 위치가 조정된다.

  • 지정된 relative 포지션의 부모가 없으면 가장 가까운 부모중에 relative 포지션을 찾아서 기준으로 삼는다
  • 아래 코드는 부모 div에 대한 postion이 static 이므로 더 상위 부모로 올라가서 body를 기준으로 삼음
  • 포지션을 지정하지 않을 경우 position: static; 이 기본 설정이다.
<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 1000px;
        margin: 50px;
      }
      div {
        width: 300px;
        height: 300px;
        background-color: wheat;
      }
      .green {
        background-color: teal;
        position: absolute;
        width: 100px;
        height: 100px;
        bottom: 20px;
        right: 0px;
      }
    </style>
  </head>
  <body>
    <div>
      <div class="green"></div>
    </div>
  </body>
</html>

만약 위의 결과와 달리 body가 아니라 감싸고 있는 div를 기준으로 안쪽에 있는 div 위치를 옮기고 싶으면, 감싸고 있는 div 즉 부모 박스에 position: relative; 코드를 더해주면 아래와 같이 결과가 바뀐다.

3.14 Pseudo Selector Part One

pseudo selector : 엘리먼트를 좀 더 세부적으로 선택해주는것을 뜻함.

  • 예를들어 span 다음에 오는 클래스를 선택하고 싶거나
  • 처음이나 마지막에 있는 클래스만 선택하고 싶은 경우 사용 가능!
  • 이 방법이 일일이 id나 class를 지정해 주는것 보다 직관적일 수 있다.
<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 1000vh;
        margin: 50px;
      }
      div {
        width: 100px;
        height: 100px;
        background-color: wheat;
      }
      div:first-child {
        background-color: tomato;
      }
      div:last-child {
        /*last-child : 리스트에 있는 div들 중 마지막을 뜻함*/
        background-color: teal;
      }
    </style>
  </head>
  <body>
    <div></div>
    <div></div>
    <div></div>
    <div></div>
    <div></div>
  </body>
</html>


앞에서 배운 first-child, last-child 외에도 nth-child()를 사용한 방법도 있다.

<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 1000vh;
        margin: 50px;
      }
      span {
        background-color: tomato;
      }
      /*
      span:nth-child(2),
      span:nth-child(4),
      span:nth-child(6),
      .. 이렇게 하는것보다 even으로 짝수만 지정해주는게 현명!
      홀수만 선택하고 싶으면 span:nth-child(odd)
      */
      /*
      span:nth-child(3n+1)
      이렇게하면 1,4,7,10,.. 순서에 해당하는 자식들이 해당됨
      */
      span:nth-child(even) {
        background-color: teal;
      }
    </style>
  </head>
  <body>
    <span>hello</span>
    <span>hello</span>
    <span>hello</span>
    <span>hello</span>
    <span>hello</span>
    <span>hello</span>
    <span>hello</span>
    <span>hello</span>
  </body>
</html>
  • nth-child() 사용시 even, odd, n3 + 1 등 다양한 방법으로 내가 원하는 엘리먼트를 선택 가능!

3.15 Combinators

<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 1000vh;
        margin: 50px;
      }
      span {
        background-color: yellowgreen;
        padding: 5px;
        border-radius: 10px;
      }
      p span {
        /*div p sapn 을써도 된다 = 원하는 만큼 부모를 앞에 나열해줘도 된다.*/
        /*p를 부모로가진 자식 span의 글자색을 white*/
        color: white;
      }
      div > span {
        text-decoration: underline;
      }
    </style>
  </head>
  <body>
    <div>
      <span>hello</span>
      <p>
        Lorem ipsum dolor sit amet consectetur adipisicing elt. Quod commodi et
        sequi ipsam odi dolorem. <span>inside</span>
      </p>
    </div>
  </body>
</html>

p span { ~ }

p를 부모로 가진 모든 span에 대해 스타일을 지정한다. 위의 코드에서 p 아래 자식 span은 하나이고 해당 글자색을 white로 설정

div > span { ~ }

이렇게 코드를 작성하면 div 아래 자식 span들 중 바로 아래(밑) 자식의 span에게만 스타일을 지정


아래 코드에서는 p+span { } 코드를 잘 살펴보자.

<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 1000vh;
        margin: 50px;
      }
      span {
        background-color: yellowgreen;
        padding: 5px;
        border-radius: 10px;
      }
      p span {
        /*div p sapn 을써도 된다 = 원하는 만큼 부모를 앞에 나열해줘도 된다.*/
        color: white;
      }
      p + span {
        text-decoration: underline;
      }
    </style>
  </head>
  <body>
    <div>
      <p>
        Lorem ipsum dolor sit amet consectetur adipisicing elt. Quod commodi et
        sequi ipsam odi dolorem. <span>inside</span>
      </p>
      <span>hello</span>
    </div>
  </body>
</html>

p + span { ~ }

이 + 기호를 사용하면 p와 형제인 span에 대한 스타일 지정이 된다. 즉 위 의 코드에서는 p 자식인 span에 대해서는 스타일 지정이 되지 않는다. 여기서 중요한건 span이 p 바로 다음에 오는 형제이기 때문에 hello에 underline을 할 수 있다. span 과 p 사이에 다른 엘리먼트가 오면 hello에 underline을 하라는 코드는 실행되지 않는다.(span이 p의 바로 뒤에 오지 않기 때문) 이때 사용할 수 있는건 ~ 이며, 이와 관련해서는 아래 3.16에서 이어서 다룬다.

3.16 Pseudo Selector part Two

e.g. span과 p가 형제인데 바로 뒤에 오지 않을때 스타일 지정을 위해 쓸수 있는것 : p ~ span { }

  • p span { } : 부모(p)와 자식(span)
  • p > span { } : 부모(p) 바로 밑의 자식(span)
  • p + span { } : p와 형제인 바로 다음에 오는 span
  • p ~ span { } : p와 형제인 span. 바로 다음에 오지 않아도 됨.
  • 그 밖의 주요 Pesudo Selector - MDN [링크]
<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 1000vh;
        margin: 50px;
      }
      span {
        background-color: yellowgreen;
        padding: 5px;
        border-radius: 10px;
      }
      p span {
        /*div p sapn 을써도 된다 = 원하는 만큼 부모를 앞에 나열해줘도 된다.*/
        color: white;
      }
      p ~ span {
        text-decoration: underline;
      }
    </style>
  </head>
  <body>
    <div>
      <p>
        Lorem ipsum dolor sit amet consectetur adipisicing elt. Quod commodi et
        sequi ipsam odi dolorem. <span>inside</span>
      </p>
      <address>hi</address>
      <span>hello</span>
    </div>
  </body>
</html>

Attribute Selector [링크]

  • id나 class를 일일이 지정하지 않고 attribute selector를 사용해서 지정할 수 있다.
<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        height: 1000vh;
        margin: 50px;
      }
      input {
        border: 1px solid wheat;
      }
      input:required {
        border: 1px solid tomato;
      }
      input[type="password"] {
        background-color: thistle;
      }
      input[placeholder~="name"] {
        /* ~ 는 placeholder에 'name'을 포함한 모든 input을 지정한것 */
        background-color: pink;
      }
    </style>
  </head>
  <body>
    <div>
      <form>
        <input type="text" placeholder="First name" />
        <input type="text" placeholder="Last name" />
        <input type="password" placeholder="password" required />
      </form>
    </div>
  </body>
</html>
