4.0 Transitions

Transition은 어떤 상태에서 다른 상태로의 “변화”를 애니메이션으로 만드는 방법

  • transition 이라는 속성은 state가 없는 요소에 붙어야 한다.
    • 즉, 아래 코드를 기준으로 a:hover{ ~ } 가 아닌 a{ ~ }안에서 transition을 지정해야 한다.
    • 만약 a:hover{ ~ } 안에 transition을 지정해 주면 마우스 오버시 작동은 하지만, 밖으로 마우스를 옮기는 순간 빠르게 스타일이 원복된다.
  • transition을 통해 변화시키고자 하는 엘리먼트는 state가 있는 요소에 들어있는 스타일 이어야 한다.
    • 즉, 아래 코드를 기준으로 a:hover{ ~ } 안에 font-size에 대한 설정이 없는데 a{ ~ }에서 transition 으로 font-size를 바꾸려 한다면 해당 명령은 수행되지 않는다.
<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      a {
        color: white;
        background-color: tomato;
        text-decoration: none; /* 밑줄 없애기 */
        padding: 3px 5px;
        border-radius: 5px;
        font-size: 60px;
        transition: background-color 3s ease-in-out, color 7s ease-in-out,
          border-radius 5s ease-in-out;
        /* tramsition: all 5s ease-in-out; <- 이렇게하면 a:hover에 있는 글자색, 배경색의 변화를 모두 5초 동안 진행한다는 의미*/
      }
      a:hover {
        border-radius: 20px;
        color: tomato;
        background-color: wheat;
      }
    </style>
  </head>
  <body>
    <a href="#">Go home</a>
  </body>
</html>

해당 결과물은 마우스오버해서 변화가 진행되는 순간을 찍은 이미지 이므로 직접 돌려보고 마우스 오버 해서 변화를 보기를 추천

4.2 Transformations

transformation은 요소를 변형시킬 수 있다.

  • transformation은 다른 요소의 box를 변형시키지 않고 원하는 요소를 이동시키기 위해 사용한다.
    • 즉, transformation은 다른 box element, 이미지에 영향을 끼치지 않는다.
<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      img {
        border: 10px solid black;
        border-radius: 50%; /*원래 사각형에서 50%를 주면 원이 된다! */
        transform: rotateY(40deg) rotateX(80deg) rotateZ(25deg) translateX(300px);
        /* y,x,z축으로 이미지를 회전, x축으로 300px 이동시킴 */
      }
    </style>
  </head>
  <body>
    <img src="noru.jpg" />
  </body>
</html>

  • transformation을 통해 노루 이미지를 y축으로 40도, x축으로 80도, z축으로 25도 돌리고 x축으로 300px (우측으로) 이동시킨 결과.
  • 위에 적힌 코드 외에도 여러가지 transformation이 가능하다.
    • transform - MDN [링크]


transformation 만 사용해서는 아무것도 아니다..! transition을 같이 사용해서 변형에 변화를 더해주면 훨씬 아름다운 모션을 만들어낼수 있다!! 아래 코드를 참고하자.

<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      img {
        border: 10px solid black;
        border-radius: 50%; /*원래 사각형에서 50%를 주면 원이 된다! */
        transition: transform 3s ease-in-out;
      }
      img:hover {
        transform: rotateZ(90deg) rotateX(180deg);
        /* z축으로 90도 x축으로 180도 돌린다. */
      }
    </style>
  </head>
  <body>
    <img src="noru.jpg" />
    <span>Look! here!!</span>
  </body>
</html>

  • 마우스 오버시 변화하는 모습을 캡쳐한 이미지로, 직접 웹페이지에서 돌려보길 강추!

4.3 Animations part One

마우스 오버해서 움직이는건 애니메이션이 아니다. 마우스가 올라가지 않아도 자동으로 움직이는 애니메이션을 만들어보자.

  • @keyframes 를 먼저 써주고, 내가 만들고 싶은 애니메이션 이름을 써준다. 아래 코드에서는 SuperSexyCoinFlip 이라는 애니메이션 이름을 사용
  • 그 후에 from{ ~ }, to{ ~ } 안에 transform 코드를 작성해 주고 애니메이션 효과를 부여할 img 에 animation: SuperSexyCoinFlip 3s ease-in-out infinite; 코드를 작성해서 어떤 에니메이션을 어떻게 적용할지 설정한다.
<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      @keyframes SuperSexyCoinFlip {
        from {
          transform: rotateY(0);
        }
        to {
          transform: rotateY(360deg);
        }
      }
      img {
        border: 10px solid black;
        border-radius: 50%; /*원래 사각형에서 50%를 주면 원이 된다! */
        animation: SuperSexyCoinFlip 3s ease-in-out infinite;
        /* infinite가 없으면 처음 웹페이지 로딩후 3초간 한번만 자동으로 움직이고 더이상 움직이지 않는다!! */
      }
    </style>
  </head>
  <body>
    <img src="noru.jpg" />
  </body>
</html>

  • 웹페이지를 열자마자 3초를 한텀으로해서 무한하게 움직이는 애니메이션을 볼 수 있다.

4.4 Animations part Two

<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        display: flex;
        height: 100vh;
        align-items: center;
        justify-content: center;
      }
      @keyframes SuperSexyCoinFlip {
        0% {
          transform: rotateY(0);
        }
        25% {
          transform: scale(2);
          opacity: 0; /* 안보이게 만듬 */
        }
        50% {
          border-radius: 0px;
          border-color: teal;
          transform: rotateY(180deg) translateY(-100px);
        }
        75% {
          transform: scale(5);
          border-radius: 20px;
        }
        100% {
          transform: rotateY(0) translateY(0px);
        }
      }
      img {
        border: 10px solid black;
        border-radius: 50%; /*원래 사각형에서 50%를 주면 원이 된다! */
        width: 100px;
        animation: SuperSexyCoinFlip 5s ease-in-out infinite;
        /* infinite가 없으면 처음 웹페이지 로딩후 3초간 한번만 자동으로 움직이고 더이상 움직이지 않는다!! */
      }
    </style>
  </head>
  <body>
    <img src="noru.jpg" />
  </body>
</html>

4.3에서는 @keyframes{ ~ } 안에 from{ ~ } 과 to{ ~ } 두가지를 이용해 애니메이션을 만들었다면, 여기서는 0% ~ 100% 로 5단계가 무한히 반복해서 움직이도록 만들어 보았다.

  • 한가지 팁은 애니메이션을 만들때 되도록이면 transform만으로 작업을 하라는 것.
    • 일부 기능은 애니메이션화 시킬때 작동되지 않는것들이 있다고 한다.
    • 예를들면 font-size..? 이런거는 되는지 안되는지 확인해 봐야한다.
  • Animista [링크] 애니메이션을 사용하기 위한 콜렉션!!

4.5 Media Queries

media query는 오직 CSS만을 이용해서 스크린의 사이즈를 알 수 있는 방법. 여기서 말하는 스크린 사이즈는 내 웹사이트를 보고 있는 사용자의 스크린 사이즈! 사용 예시를 하나 들어보자면 아래와 같다.

@media screen and (min-width: 600px) and (max-width: 750px) { ~ }

이 경우 사용자의 스크린이 600px ~ 750px인 경우에 대한 스타일 지정이다.

  • @media 다음에 screen 은 스크린에 대한 설정이다. 라고 이해하면 된다.
    • screen 외에도 print는 웹페이지를 어떻게 출력할지 설정할 수 있다.


<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        display: flex;
        height: 100vh;
        align-items: center;
        justify-content: center;
      }
      div {
        width: 200px;
        height: 200px;
        background-color: teal;
      }
      @media screen and (max-width: 600px) {
        div {
          background-color: tomato;
        }
      }
      @media screen and (min-width: 601px) and (max-width: 800px) {
        div {
          background-color: wheat;
        }
      }
      @media screen and (min-width: 1200px) {
        div {
          background-color: turquoise;
        }
      }
    </style>
  </head>
  <body>
    <div></div>
  </body>
</html>

위의 media query를 풀이해 보자면

  • 600px 이하 이면 div는 tomato색
  • 601px 이상 800px 이하 사이즈 이면 div는 wheat색
  • 800px 보다 크고 1200px 미만이면 div는 teal색
  • 1200px 이상 이면 div는 turquoise


<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        display: flex;
        height: 100vh;
        align-items: center;
        justify-content: center;
        flex-direction: column;
      }
      div {
        width: 200px;
        height: 200px;
        background-color: teal;
      }
      @media screen and (min-width: 601px) and (max-width: 1200px) and (orientation: landscape) {
        /* orientation: landscape > 화면이 가로일 경우
           orientation: portrai > 화면이 세로일 경우  */
        div {
          background-color: wheat;
        }
      }
      span {
        font-size: 36px;
      }
      @media screen and (orientation: landscape) {
        span {
          display: none; /* 가로모드일 경우 span을 보여주지 않도록 설정 */
        }
      }
    </style>
  </head>
  <body>
    <div></div>
    <span> Please flip your phone :( </span>
  </body>
</html>

  • 기본적으로 div의 배경은 teal로 설정되어 있지만
  • 화면 사이즈가 601px ~ 1200px이고 가로모드인 경우에 wheat 색이면서 span 을 보여주지 않도록 설정

4.6 Media Queires Recap

media query는 해당 조건을 만족하면? 지정된 CSS를 실행하라는 것!

  • and 를 써서 조건을 연결한다
  • mini-device-width, max-device-width 도 있는데 이는 오직 핸드폰에만 적용된다 [device-width]
    • 따라서, 이걸쓰면 데스크톱에서 브라우저는 해당 코드를 이해할 수 없다.
  • [Using media queries] : aspect-ratio을 사용하면 스크린이 레티나 디스플레이 인지 아닌지.. 등까지 고려한 구현 가능. 다양한 방법이 있으니 살펴보자!
<!DOCTYPE html>
<html lang="kr">
  <head>
    <title>Home - my first website</title>
    <style>
      body {
        display: flex;
        height: 100vh;
        align-items: center;
        justify-content: center;
        flex-direction: column;
      }
      div {
        width: 200px;
        height: 200px;
        background-color: teal;
      }
      @media screen and (min-width: 601px) and (max-width: 1200px) and (orientation: landscape) {
        /* orientation: landscape > 화면이 가로일 경우
           orientation: portrai > 화면이 세로일 경우  */
        div {
          background-color: wheat;
        }
      }
      span {
        font-size: 36px;
      }
      @media screen and (orientation: landscape) {
        span {
          display: none; /* 가로모드일 경우 span을 보여주지 않도록 설정 */
        }
      }
      @media print {
        body {
          background-color: tomato;
        }
      }
    </style>
  </head>
  <body>
    <div></div>
    <span> Please flip your phone :( </span>
  </body>
</html>

4-5에서 @media print { ~ } 에 대해 잠깐 언급했는데, 4-5 html코드에서 @media print { ~ } 부분만 추가한 코드이다. 이는 프린트될때 화면이 어떻게 출력될지를 설정해주는 방법이며 미디어 쿼리에서 설정한 바와 같이 배경을 tomato 색으로 출력하게 된다. (출력시 우측 탭에 ‘배경 그래’ 부분이 체크되어 있어야 함.)
