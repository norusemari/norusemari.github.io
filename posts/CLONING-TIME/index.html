  1. 안에 style.css를 불러오는 링크를 만들려면 link:css를 입력 후에 엔터를 누르면 자동완성된다. - > **자동으로 완성된 링크 태그** : < link rel="stylesheet" href="style.css"> - 만약 default로 완성된 css파일 경로가 특정 폴더 안에 있다면 **href=폴더명/style.css로 수정**
  2. 다양한 폰트 적용하기
    • 1안) css 파일에서 body{ ~ } 안에 font-familt: 까지 치면 고를 수 있다.
    • 2안) [Google Fonts]에 접속후 원하는 폰트 클릭후 우측탭에서 나오는 link나 @import를 선택하여 아래 나오는 코드 복사해서 사용
      • @import 는 css에서 폰트를 적용하는 거고
      • link는 html코드 내의 head에서 폰트를 적용하는거다.
    • 주의 꼭 필요한 폰트만 추가하자! 안그러면 웹사이트가 느려진다.

  1. Reset CSS [링크]
    • 기본적으로 브라우저에서 제공하는 body의 margin을 없애는 코드.
    • 링크에서 해당 코드를 다운받고, reset.css 파일을 만들어서 style.css 파일안에 @import “reset.css”; 해라. html파일에서 link하면 지저분하니까 style.css 파일에 모아두는 것.
    • 이걸하는 이유는, 브라우저 스타일을 제공하고 우리가 원하는 스타일을 만들기 위함
  2. 디자인 시안을 구현할때 정확한 간격(px)을 측정하기 위해서는 맥에서 마우스로 측정할 수도 있고, page ruler redux 라는 프로그램을 크롬에 깔아서 측정해도 된다.


  1. :not() 무언가 적용되지 않았으면 할때 사용한다.

    • #login-form input:not([type=”submit”]) { border-bottom: 1px solid rgba(0, 0, 0, 0.2); transition: border-color 0.3s ease-in-out; }

    • 이코드 보다 앞에서 login-form input에 대해 일괄 적용한 내용 중, 일부 input(여기서는 type=”submit”인 경우)에 대해 적용을 원치 않는 사항을 명시
    • [] 안에 쓴 코드는 attribute selector(특성 선택자)라 한다.
  2. from 에는 크게 두 가지 powerful한 속성(attribute)이 있다. 하나는 action이고 다른 하나는 method다.
    • action은 어떤 페이지로 data를 보낼건지 지정할 수 있다.
    • form 태그에서 action=”&&&.html” 을 적을 경우, 반드시 &&&.html 파일이 존재하고 있어야 한다. - method 2가지 방식 중 하나를 쓸 수 있다. POST와 GET
    • post는 백엔드 서버에 정보를 전송하는 방식.
    • get은 보안에 취약하므로, userid나 password를 get방식으로 전송하면 안된다.
      • 보통 url에 포함해도 상관없는 정보들을 get방식으로 전송한다.

    • < form action=”friends.html” method=”get” id=”login-form”>

      • friends.html 페이지로 data를 보낸다. username, password 입력후에 login을 누르면 friends.html 파일이 열리고 url 끝부분에서 로그인 창에서 입력한 username과 password를 확인할 수 있다.

      • file:///Users/noru.semari/Documents/kokoa_clone/kokoa-clone-2020/kokoa-clone-2020/friends.html?username=test1232&password=pas1234

  1. CSS 에서 box-sizing: border-box; 의 의미는 내가 padding 값으로 얼마를 주던지 간에 본래의 박스크기를 유지하라는 뜻이다.
    • 즉, width : 200px인 박스에 padding-left : 50px를 설정하면 패딩도 주면서 너비를 유지하기 위해 결국 250px 가된다.
    • 하지만, 내가 원하는것이 원래의 박스 너비를 그대로 유지하는 것 이라면 css로 아래와 같이 명시해주면 된다
